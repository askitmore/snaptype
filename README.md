# Snaptype

A modern, feature-rich Markdown editor built with Electron, React, and AI-powered chat capabilities.

## Features

- **Rich Markdown Editing**: Built with Tiptap editor, supporting headings, lists, tables, and more
- **AI-Powered Chat**: Integrated AI assistant powered by Google's Gemini API
- **File Management**: Save, open, and manage `.md` files with automatic file association
- **Slash Commands**: Quick formatting with `/` command menu
- **Keyboard Shortcuts**: Efficient editing with shortcuts like `Cmd+S` to save, `Cmd+1/2/3` for headings
- **Multi-Window Support**: Open multiple documents in separate windows
- **Unsaved Changes Detection**: Warns before closing with unsaved changes
- **Document Conversion**: Import Word (`.docx`) and PDF files directly
- **Modern UI**: Clean, minimal interface with customizable chat sidebar

## Keyboard Shortcuts

- `Cmd/Ctrl + S`: Save file
- `Cmd/Ctrl + Shift + S`: Save as
- `Cmd/Ctrl + O`: Open file
- `Cmd/Ctrl + N`: New window
- `Cmd/Ctrl + 1/2/3`: Toggle heading levels
- `/`: Open command menu
